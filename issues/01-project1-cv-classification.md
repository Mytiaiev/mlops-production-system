# Project 1: CV Classification System

**Phase:** 1 - Basic MLOps + Computer Vision  
**Timeline:** Months 1-6  
**Priority:** High

## Objective

Build a production-ready image classification system with MLOps best practices, including experiment tracking, model serving via API, containerization, and basic monitoring.

## Project Overview

**Use Case:** Image classification (e.g., product categorization, defect detection, or custom dataset)

**Tech Stack:**
- PyTorch + torchvision for model training
- MLflow for experiment tracking and model registry
- FastAPI for serving predictions
- Docker for containerization
- Loguru for structured logging

## Technical Requirements

### Model Performance
- Validation accuracy: >85%
- Inference latency: <100ms per image
- Model size: <100MB

### API Requirements
- Response time: <200ms (including preprocessing)
- Support for JPEG, PNG image formats
- Input validation and error handling
- API documentation (auto-generated by FastAPI)

### Code Quality
- Test coverage: >80%
- Type hints on all functions
- Linting: pass black, ruff, mypy
- No hardcoded values (use configs)

### Documentation
- README with setup instructions
- API documentation
- Architecture diagram
- Deployment guide
